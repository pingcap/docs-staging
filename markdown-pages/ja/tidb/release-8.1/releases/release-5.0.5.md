# TiDB 5.0.5 リリースノート {#tidb-5-0-5-release-note}

発売日: 2021年12月3日

TiDB バージョン: 5.0.5

## バグ修正 {#bug-fix}

-   ティクヴ

    -   `GcKeys`タスクが複数のキーによって呼び出されたときに動作しない問題を修正しました。この問題により、コンパクション フィルター GC が MVCC 削除情報を削除しない可能性があります[＃11217](https://github.com/tikv/tikv/issues/11217)
